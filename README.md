# Portfolio

My portfolio build [Jekyll](https://jekyllrb.com/). See me in action at <https://rsseau.fr>.

![screenshot](./public/screenshot.png)

## Setup

```bash
bundle install
```

Then start development server

```bash
jekyll serve
```

or build website

```bash
jekyll build
```
