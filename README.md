# Portfolio

My portfolio build [Gatsby](https://www.gatsbyjs.com/). See me in action at <https://rsseau.fr>.

![screenchot](./public/screenshot.png)

## Setup

```sh
npm install
```

Then start development server

```sh
npm run develop
```

or build webiste

```sh
npm run build
```
